# ❌⭕ Tic Tac Toe Game (JavaScript Project)

A simple and interactive Tic Tac Toe game built using HTML, CSS, and JavaScript. Players can take turns and the game automatically detects the winner.

---

## 📌 Project Description

This is a web-based Tic Tac Toe game where two players can play against each other. The game checks all winning combinations and displays the winner dynamically.

---

## 🚀 Features

* Two-player game (X vs O)
* Automatic turn switching
* Winner detection using predefined patterns
* Reset and New Game functionality
* Interactive UI with color styling
* Fully responsive layout

---

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript

---

## 📂 Project Structure

* `Tic_tac_toe.html` → Main structure
* `Tic_tac_toe.css` → Styling
* `Tic_tac_toe.js` → Game logic

---

## ▶️ How to Run

1. Download or clone this repository
2. Open `index.html` in any browser
3. Start playing by clicking on boxes
4. Use **Reset Game** or **New Game** buttons to restart

## 💡 How It Works

* Each box click registers a player's move (X or O)
* Turn alternates between players
* Winning patterns are stored in an array
* The game checks combinations after every move
* When a match is found, the winner is displayed
